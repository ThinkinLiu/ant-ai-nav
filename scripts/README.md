# 数据库管理脚本

本目录包含数据库管理和维护相关的脚本。

## 📜 脚本列表

### prepare.sh
**用途**: 项目准备脚本，在开发前执行

**使用方法**:
```bash
bash scripts/prepare.sh
```

**功能**:
- 检查并创建必要的目录
- 设置环境变量

### dev.sh
**用途**: 启动开发服务器

**使用方法**:
```bash
bash scripts/dev.sh
```

### build.sh
**用途**: 构建生产版本

**使用方法**:
```bash
bash scripts/build.sh
```

### start.sh
**用途**: 启动生产服务器

**使用方法**:
```bash
bash scripts/start.sh
```

## 📦 数据迁移

推荐使用管理后台的数据迁移功能，无需手动执行脚本：

### 导出数据
1. 访问 `/admin/data-migration`
2. 选择导出模式：
   - **全部导出**: 包含所有数据（含用户信息）
   - **业务数据**: 不含用户信息（推荐迁移）
   - **内容数据**: 核心内容（工具/资讯/名人堂等）
   - **设置数据**: 系统配置
   - **自定义**: 手动选择表
3. 点击导出下载JSON文件

### 导入数据
1. 访问 `/admin/data-migration`
2. 选择导入模式：
   - **合并模式**: 保留现有数据，更新/新增
   - **替换模式**: 清空现有数据后导入
3. 上传JSON备份文件

## 🔧 故障排除

### 问题: 数据库连接失败
**解决**: 检查 `.env.local` 中的 Supabase 配置是否正确

### 问题: 导入失败
**解决**: 
1. 检查JSON文件格式是否正确
2. 尝试使用合并模式而非替换模式
3. 查看管理后台日志
